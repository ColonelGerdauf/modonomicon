# Empty Page

![Empty Page](/img/docs/basics/page-types/empty-page.png)

**Page type:** `modonomicon:empty`

Displays an empty page. Not really sure what you may want to use that for, but here we are.

## Attributes

None.

## Usage Examples

`<entry>.json`:

```json
{
  ...
  "pages": [
    {
      "type": "modonomicon:empty",
      "anchor": ""
    }
  ]
}
```
